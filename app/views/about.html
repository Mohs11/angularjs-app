This App collates and displays data about football clubs across Europe. You're able to filter this information by League, and the search function allows you to search for any attribute including: name, ground, location and stadium capacity.

I created this app to explore and learn Angular js. I created this in September 2017 and sourced the data from Wikipedia. This app does not use an API and so the information is currently static. I hope to attach it to an API soon. The icons have been sourced from Font Awesome 4. The club logos have been sourced from the club websites, I do not own these logos and they belong to the clubs.
